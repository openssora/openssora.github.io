七、Codex 常见的错误和解决方法

7.1 Codex 401 Unauthorized 错误

Codex 出现：401 Unauthorized 的错误，就是授权到期了，或者会员过期了，你只需要重新授权就行。

在终端执行：/logout 退出 Codex 之后，再重新执行 Codex 就可以进行二次授权。

7.2 Codex 502 stream error 或者 re-connect 失败

如果 Codex 出现502 或者 重新链接 connect fail 这些错误，大部分是 网络问题，你可以重新开启你的魔法为全局的 Tun 模式。

或者开系统代理之后，再在终端上 export HTTPS_PROXY=127.0.0.1:7890 就行。
