如何访问使用OpenAI o1 模型

目前，o1模型已经逐步向所有ChatGPT Plus和 Team用户开放，未来会考虑对免费用户开放。

分为两个模型，o1预览版 和 o1 mini，o1-mini就是更快更小更便宜，推理啥的都不错，极度适合数学和代码，就是世界知识会差很多，适用于需要推理但不需要广泛世界知识的场景。

o1预览版使用限制30条/每周，o1-mini的50条/每周。

从次数的限制来看，o1的成本比GPT-4o贵了整整N倍，GPT-4o是80条/3小时。

对于个人用户，访问使用的话，直接在右上角切换模型就行，但是o1只给ChatGPT Plus会员推送。

如果你不知道怎么升级订阅ChatGPT Plus 会员的话，可以参考教程： https://upchatgpt.cn/how-upgrade-chatgpt-plus/

或者自己开通一张虚拟卡进行升级ChatGPT Plus也行，使用wildcard虚拟卡就可以：https://yeka.ai/i/UPGPT

关于API等级的说明：

5级 API 用户,也就是说要消费到1000刀才能满足！

  • o1-preview: 旨在解决跨领域难题的推理模型
  • o1-mini: 更快、更便宜的推理模型，特别擅长编码、数学和科学。

如何使用支付宝升级订阅ChatGPT Plus 会员

（1）首先进入wildcard虚拟卡官网之后，选择 『立即开通』，这个才是虚拟卡，旁边的是套壳服务，有兴趣的可以体验。

（2）选择 『立即开通』 之后，就可以跳到 『我的卡片』 页面，进行开通虚拟卡就行，选择2年的就可以。 输入邀请码 UPGPT 是可以优惠开卡的，有很多朋友不知道这个邀请码，所以比较贵了！

支付时，是使用支付宝的方式，同时需要你实名开卡，这也是避免其他坏人用来xi钱！

（3）开完虚拟卡之后，就会得到跟下面一样的卡片，这个卡片就可以用来绑定海外的服务进行支付。

（4）有了卡片之后，就可以使用『支付服务』了，选择自己需要支付的服务，然后点击下一步，下一步其实就是往你的虚拟卡上充值金额。

（5）比如升级ChatGPT需要20刀，那就充20刀，不过建议充值21刀，因为部分服务绑定虚拟卡时是需要扣除0.01刀费用，然后再退回的。

（6）充值之后，就可以拿卡片的信息，到海外的服务进行进行绑卡消费啦！如果还不知道怎么操作，可以联系客服！

「另外，卡片余额可以直接秒提现到支付宝！」
