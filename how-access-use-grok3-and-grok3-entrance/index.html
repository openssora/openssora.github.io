下面介绍一下Grok3模型，以及如何访问和使用Grok3的教程，Grok3的访问入口在哪里等。 和 使用Grok3的网页版官方地址入口。

Grok3 分为两个模型： Grok-3 Reasoning Beta 和Grok-3 mini Reasoning。 同时 Grok3 还支持一个新的模式：大脑模式（Big Brain Mode）。

Grok3 还是 和 o1 一样的 Reasoning 模型，同样支持 Deep Search （深度研究）功能。

如何体验 Grok3呢？

Grok 3仅对X上的Premium+用户开放。你需要订阅X的Premium+服务才能访问Grok 3的功能。

可以申请一张虚拟信用卡进行订阅, 平台上有教程。申请地址：

https://wildscard.com

最新补充

2025/02/20 更新：

Grok 3 已经免费开放，不需要订阅X的Premium+服务。 需要更多次数，订阅Premium、Premium+、以及SuperGrok都可以提高次数。

性价比较高的是：X平台上的Premium 服务（8美元）。 深度搜索次数是免费的Grok3的12倍。

教程参考：https://openssora.com/how-increase-grok3-think-search-times

也就是说，你需要注册一个X账号（原推特）。并且升级到对应的版本才能有权限访问 Grok 3。

目前，Grok Web网页版 在英国/欧盟仍然无法访问。 但是你可以在 IOS 的 APP Store 上进行安装 。

教程开始之前，我们先来了解一下 Grok 的 网页版官方使用地址：

web网页版：https://grok.com/

Grok不仅在官网上使用，你也可以在 X（原推特）上进行使用。打开 X 之后，（通常在界面底部或侧边栏）上就能看到 Grok。

X （原推特）上的网页版：https://x.com/i/grok

而需要订阅升级“Premium +”的话，进入 X 官网之后，点击侧边栏上的 “Premium”，就可以进入订阅页面。

按年订阅有优惠，不过正常我们都是按月进行订阅。费用：40 $。

在点击下方 “Subscribe & Pay” 之前，我们先申请一张虚拟信用卡，再进行订阅和支付。

因为外国人的消费习惯，都是先预支的。所以大部分付款都是使用信用卡。而国内用户，大部分也没有master、visa国际信用卡。

但是我们可以借助 “master 虚拟信用卡” 解决。

虚拟卡申请：https://wildscard.com

虚拟信用卡平台，支持订阅支付的 X 和 其他平台服务。

申请之后，会得到对应的卡号和账单地区等。

虚拟卡的余额可以通过「支付宝」进行充值，不使用的余额也可以直接体现到「支付宝」。

而这次订阅 X 的 Premium + 服务，需要 40刀。 所以你充值40刀就行。 不需要充太多。

为什么使用虚拟信用卡？因为 X 属于海外的，订阅时，有时候无法使用国内的方式进行支付。

在订阅页面上，把我们的虚拟信用卡的信息填入之后，订阅即可。你也可以观看平台上的教程。记得选中 “Premium +” 服务，才有 Grok 3 权限。

进行绑卡支付时，如果失败了，是因为网络上的风控。记得使用跟虚拟卡同样地址，或者其他网络环境比较顺畅的魔法节点 进行重试。

注意，如果后面不需要续订的话，记得到 X 上，找到 Premium 的Manage上，进行取消续订。

订阅 Premium + 成功之后，你就可以使用 Grok 3 哦。你可以到 Grok 网页版上，登录时，使用 X 账号进行登录。

或者到 X 上，的侧边栏上的 Grok 上使用 Grok 3。
